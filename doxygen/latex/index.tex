\begin{DoxyParagraph}{The sample E\-S\-P8266 with ili9341 display project }
See \hyperlink{md_COPYRIGHT}{C\-O\-P\-Y\-R\-I\-G\-H\-T.md} for a full copyrrite notice for the project
\end{DoxyParagraph}
This project uses a E\-S\-P8266 to drive an ili9341 display I wrote the following support functions C\-O\-R\-D\-I\-C C table generator and 3\-D transformation code support functions Wireframe viewer and C generator for Earth coastline data Small Print with full floating point support -\/ optional B\-D\-F font to C conversion code and optimized display code I\-L\-I9341 display driver code

I Updated\-: H\-S\-P\-I code that correctly handles memory size and agligment

Two demos are provided
\begin{DoxyItemize}
\item C\-U\-B\-E demo
\item Wireframe Earth viewer -\/ still neads hiden line removal
\end{DoxyItemize}
