uint8_t  parameter 
)

Read ILI9341 16bit register value Special undocumented way to read registers on 4 wire SPI configurations See M:[0-3] control bits in ILI9341 documenation.

  • refer to interface I and II modes. Most ILI9341 displays are using interface I rather the interface II mode. It appears that direct reading has been disabled on most SPI displays.
    Parameters
    [in]regregister to read
    [in]parameterparameter number to read
    Returns
    16bit value
