12 @par Description
13  * ESP8266 ILI9341 with multiple window support WEB server and other networking demos
14  * The specific display is an TM022HDH26 display I got from ebay.
15  * I wrote the following support functions from scratch
16  * Font compiler and display code
17  * ILI9341 display driver with multiple custom sized window support
18  * (I rewrote most of Adafruit code ili9341 graphics library)
19  * readPixel() and custom sized window scrolling
20  * Not limited by ILI9341 hardware scroll restrictions.
21  * CORDIC C table generator and 3D transformation code support functions
22  * Wire frame viewer and C generator for Earth coastline data
23  * I created two wireframe demos
24  * Cube wireframe dataset
25  * Earth coastline dataset - wireframe view still needs hidden line removal option
26  * Small Printf with full floating point support along with ftoa/atof and integer conversions
27  * BDF fonts and a BDF font to C conversion code and optimized display code
28  * WEB server using SD CARD with CGI processing - files and CGI results can be ANY SIZE!
29  * WEB server can update TFT display
30  * Network server client example for display updates
31  * Uart network server client for serial uart to network bridge.
32  * Generic queue handling code
33  * Uart driver.
34  * POSIX wrappers for FatFS support by ChaN
35  * POSIX time functions and RTC set with NTP
36  * Timers used by rtc and time functions
37  * HSPI code that can handle multiple chip select and clock frequencies
38  * Includes FatFS SD card support by ChaN
